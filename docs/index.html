solana-tools
Open-source Solana CLI toolbox built by Kevin Smits (@smitskecbs).
All tools are written in TypeScript and run from this monorepo.
Repo:  github.com/smitskecbs/solana-tools
TypeScript Solana web3.js Solana Kit (v2) Helius / custom RPC ready

How to use

Clone the repo, install once, then run any tool you like from your terminal. For best performance always use your own RPC (e.g. Helius).

Quick start
Clone & install
git clone https://github.com/smitskecbs/solana-tools.git cd solana-tools npm install
Recommended: set your RPC
RPC_URL="https://mainnet.helius-rpc.com/?api-key=YOUR_HELIUS_KEY"

Available tools

Each tool lives in packages/<tool-name> with its own README and examples. Below is a quick overview plus the main command to run each one.

holder-info SPL holders overview

Inspect SPL token holders: token accounts, balances and top holders for any mint. Useful for supply analysis, whale tracking and early research.

web3.js TypeScript RPC required
Run
npm run dev -w ./packages/holder-info -- <MINT>
token-info On-chain token metadata

Fetch SPL mint info (decimals, supply, authorities) plus metadata (name, symbol, URI) directly from chain.

web3.js Metadata
Run
npm run dev -w ./packages/token-info -- <MINT>
wallet-info Wallet overview

Show SOL balance, SPL token holdings and basic wallet activity. Good for quick wallet inspection.

web3.js Balances
Run
npm run dev -w ./packages/wallet-info -- <WALLET_ADDRESS>
nft-info NFT / DAS inspector

Inspect any NFT or digital asset using Helius DAS or Solana Kit. Ideal for checking collections and single assets.

Helius DAS NFTs
Run
npm run dev -w ./packages/nft-info -- <ASSET_ID>
snapshot-airdrop Holder snapshot → CSV/JSON

Snapshot SPL token holders into CSV + JSON, with min-balance filter and exclude list. Perfect for fair airdrops and reward distributions.

Solana Kit Airdrops
Run
npm run dev:kit -w ./packages/snapshot-airdrop -- --mint <MINT> --min 1 --out snapshots
raydium-analyzer Raydium pool inspector

Query DexScreener for a given SPL mint and show Raydium pools, liquidity, 24h volume and price change.

DexScreener API Raydium
Run
npm run raydium:analyze -- --mint <MINT>
token-safety-check Token risk overview

Combines on-chain mint data, holder concentration and Raydium liquidity into a quick safety summary for any SPL token.

Mint authority Liquidity Concentration
Recommended (with your RPC)
RPC_URL="https://mainnet.helius-rpc.com/?api-key=YOUR_HELIUS_KEY" \ npm run token-safety -- --mint <MINT>
