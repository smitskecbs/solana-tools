Solana Token Safety Viewer
Web UI on top of the solana-tools CLI by Kevin Smits (@smitskecbs). Enter a mint to see authorities, holder concentration and Raydium liquidity – like a lightweight Solscan for devs & degen researchers.
Repo: github.com/smitskecbs/solana-tools
Check a token
Paste any SPL mint on Solana mainnet. The backend uses your configured Helius RPC (server-side), so it works even if public RPCs block the browser.
